📋 Setup & Deployment Guide

›
1

Open Google Apps Script

Go to script.google.com → New Project → Name it HOB CertBot

2

Paste Code.gs

Delete default myFunction() → Paste the generated Code.gs content

3

Create HTML File

Click + → HTML → Name it exactly CertificateSelector → Paste the HTML content

4

Grant Permissions

Run openCertificateUI() from Code.gs → Review & allow all OAuth scopes (Sheets, Slides, Drive, Gmail)

5

Add Menu Trigger

The onOpen() function auto-creates HOB Certificates menu in any linked Sheet

6

Test Flow

Open linked Sheet → HOB Certificates → Generate Certificates → Select 1 course + 1 employee → Generate
